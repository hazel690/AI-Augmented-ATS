# AI-Augmented-ATS
An AI-powered recruitment engine using NLP to rank 10,000+ candidates.
This project sits at the intersection of two things I care about — people and data. Recruitment is deeply human, but the screening process is often where good candidates get lost to noise. I wanted to prove that a small, well-designed tool can make that process meaningfully better.
It's also my way of learning by doing — turning the skills I'm building as an aspiring HR Analyst into something real and useful.
 Working prototype — open to feedback and collaboration.
 

### 📌 Project Overview
As part of my target to become an **HR Analyst**, I developed this tool to solve the "Keyword Gap" in traditional recruitment. It uses **Semantic Search** to understand the context of a Job Description rather than just matching words.

### ✨ Key Features
- **Scalability:** Successfully ranks 10,000+ candidate records in real-time.
- **NLP Logic:** Uses the `all-MiniLM-L6-v2` transformer model for deep textual understanding.
- **Accuracy:** Validated at **80% accuracy** through manual HR audit samples.
- **Interactive UI:** Built with **Streamlit** to provide a user-friendly dashboard for recruiters.

### 🛠️ Tech Stack
Python, Streamlit, Pandas, Sentence-Transformers (PyTorch).
